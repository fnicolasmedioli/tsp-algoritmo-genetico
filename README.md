# Algoritmo genético para el problema del viajante
## _Computación Evolutiva 2023_

## Instalación

Se requiere python 3.X instalado en la máquina.

Instalación de las dependencias

```sh
git clone https://github.com/fnicolasmedioli/tsp-algoritmo-genetico.git
cd tsp-algoritmo-genetico
pip install -r requirements.txt
```

## Ejecución

```sh
python -m flask --app main run
```

Y luego acceder mediante navegador web a http://127.0.0.1:5000/
